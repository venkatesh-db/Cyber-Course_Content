
You are a Principal Cybersecurity Architect designing a production-grade SOC and Fraud Detection platform for a large financial payments company similar to Fiserv.

The system processes high-volume financial transactions and API gateway traffic for banks, payment processors, and merchant services.

The platform must analyze payment logs, API gateway logs, and authentication logs to detect both cybersecurity attacks and financial fraud attempts.

SYSTEM CONTEXT

The company processes:

* Millions of payment transactions per hour
* API requests from merchants, POS terminals, mobile banking apps, and partner banks
* Authentication events for customers and employees

Primary risks include:

* Payment fraud
* Credential stuffing
* Account takeover
* Card testing attacks
* API abuse
* Insider threats
* Bot attacks on payment endpoints

The SOC platform must automatically detect and classify incidents using rule engines, machine learning, and LLM-assisted reasoning.

---

3. MACHINE LEARNING ANOMALY DETECTION
